# turbo-doodle

### git使用规则：

* 暂时只用main分支，以后可能加分支和tag

  

### 作业上传途径：

- 第一种：QQ给我，我传到github

- 第二种（建议,需要github账号和基础的git知识）：

  1. clone github项目的main分支到本地；

  2. 在“/assignments/”目录下，添加自己的学号文件夹，放进作业文件；

     * 形成“/assignments/【学号】/xxx”等文件

     * 注意不要删改项目内、学号目录以外的文件

  3. commit含有作业的本地文件，推送到github上的main分支
